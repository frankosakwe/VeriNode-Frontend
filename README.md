# VeriNode — Frontend

Next.js web application for VeriNode: a decentralized savings circles (ROSCA) protocol on Stellar Soroban.

## Features

- **Savings Circles**: Create and join decentralized ROSCA savings groups
- **Collateral Management**: Manage collateralized participation in savings circles
- **Quadratic Voting**: Participate in governance with quadratic voting mechanisms
- **Emergency Safeguards**: Access emergency withdrawal and leniency voting features
- **Transaction History**: View deposit, payout, and savings history

## Tech Stack

- [Next.js](https://nextjs.org/) — React framework
- [TypeScript](https://www.typescriptlang.org/) — Type safety
- [Tailwind CSS](https://tailwindcss.com/) — Styling
- [Stellar Soroban](https://stellar.org/) — Smart contract integration

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.
